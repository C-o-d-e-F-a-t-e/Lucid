Lucid is a comprehensive web application that detects manipulated or AI-generated content across multiple formats. The platform analyzes mainly historical texts, images, and documents using advanced machine learning models to determine authenticity. Users can upload files for real-time analysis and receive detailed reports with confidence scores and detection rationale. A key feature allows users to upload their own custom datasets to train personalized detection models tailored to specific content types or domains, enhancing accuracy for specialized use cases. The system provides transparent scoring and explanatory insights for all predictions.
